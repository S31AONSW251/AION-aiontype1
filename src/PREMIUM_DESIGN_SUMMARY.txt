1. CONSISTENCY
   Apply the same styles across all similar components.
   Never bypass the design system for custom styles.

2. CLARITY
   Premium doesn't mean complex. Keep interfaces clean.
   Use whitespace effectively.

3. DEPTH
   Use shadows, glows, and layers for visual hierarchy.
   Create visual separation between elements.

4. MOTION
   Smooth transitions enhance user experience.
   Keep animations under 500ms.
   Reduce motion for accessibility.

5. LUXURY
   Quality over quantity.
   Every pixel should feel intentional.
   Attention to detail matters.

6. ACCESSIBILITY
   Premium design must be accessible to everyone.
   Test with keyboard, screen readers, and other tools.
